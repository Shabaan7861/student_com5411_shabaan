#Project Overview

this project demonstates how the infrastructure automation can be implementedusing **Powershell Desired stateconfiguration  (DSC)**.
the system configuration is defined as code and applied automatically using a build script. Validation tests are thenused to confirmthat the configurationhas been applied successfully
The project also generates system evidence to show the environment configurations and system state.

---
#Technology used

The following technologies used in this project:

- Windows Server
- PowerShell 
- PowerShell Desired state configurations (DSC)
- Pester testing framework
- Git and Github for version Control
